## Solution
```
처음엔 입력을 (select email from prob_phantom where no=1)을 했더니 안나옴. 
내 서버에서 해보고 오류난 부분을 알아보니 
Mysql Error 1093 You can't specify target table 테이블명 이렇게 나옴. 

이는 서브쿼리에 동일한 테이블을 사용할 경우 발생하는 오류임.

MySQL 은 Oracle 과는 달리 UPDATE 나 DELETE 시 
자기 테이블의 데이터를 바로 사용 못하므로 1093 에러가 발생함.

해결방법은 테이블에 별명을 지어주면 됨.

따라서 입력값은 다음과 같음.
```
